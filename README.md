# Program Pengolahan Nilai Mahasiswa

Program ini dibuat untuk memenuhi UAS
Algoritma dan Pemrograman Lanjutan.

## Fitur
- Input data mahasiswa
- Tampilkan, ubah, hapus data
- Pengolahan nilai
- Faktorial dan Fibonacci (rekursif)

## Cara Compile
gcc main.c -o nilai

## Cara Menjalankan
./nilai
