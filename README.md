# Program Pengolahan Nilai Mahasiswa (Bahasa C)

Program ini dibuat untuk memenuhi tugas Ujian Akhir Semester
Mata Kuliah Algoritma dan Pemrograman Lanjutan.
Program dibuat menggunakan bahasa C dan dijalankan melalui console/terminal.

## Tujuan Program
Program ini bertujuan untuk membantu pengolahan data nilai mahasiswa
dalam satu kelas, meliputi:
- Menyimpan data mahasiswa
- Menampilkan data mahasiswa
- Mengubah dan menghapus data mahasiswa
- Mengolah nilai (rata-rata)
- Menampilkan nilai tertinggi dan terendah
- Menampilkan fitur rekursif

## Struktur Repository
File dan folder yang terdapat dalam repository ini:
- `main.c` → File program utama bahasa C
- File screenshot (`.png`) → Bukti hasil eksekusi program

## Cara Compile dan Menjalankan Program
1. **Compile program:**

## Cara Compile
gcc main.c -o nilai

## Cara Menjalankan
./nilai

##  Menu Program
Program memiliki menu utama sebagai berikut:
1. Tampilkan seluruh data mahasiswa  
2. Tambah data mahasiswa  
3. Ubah nilai mahasiswa berdasarkan NIM  
4. Hapus data mahasiswa berdasarkan NIM  
5. Hitung dan tampilkan hasil pengolahan nilai  
6. Fitur rekursif (faktorial dan Fibonacci)  
7. Keluar dari program
